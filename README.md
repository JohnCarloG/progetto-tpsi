Realizzare il sito web di una città a scelta.

Formare un gruppo di lavoro composto da 3 o 4 membri e nominare un capo progetto (che si occuperà sia di comunicare per conto del gruppo che di creare e consegnare il repository del lavoro).

Tecnologie da utilizzare
Html
CSS
Javascript*
Bootstrap*
Github *
(quelle contrassegnate dall'asterisco saranno oggetto di valutazione).

Consegnare anche una relazione che riporti l'organizzazione del lavoro in tutte le sue fasi.

______ IMPORTANTE ______

Deve esserci almeno una pull request per ciascun componente del gruppo.

Tutti devono programmare e fare commit, saranno prese in considerazione le linee di codice scritte da ognuno.

Bisogna essere a conoscenza di tutto ciò che verrà consegnato, anche di parti non scritte direttamente da voi (dichiarazioni tipo "non lo so, non l'ho scritto io" comporteranno una valutazione pari a 2).

Utilizzo dell'IA generativa: come al solito è consentito, ma non saper spiegare nel dettaglio ciò che si è copia-incollato comporterà una penalità di 4 punti sulla valutazione di ciascun membro del gruppo. Vi suggeriamo quindi di non abusarne.

Consegna in ritardo: -3 punti di penalità per ciascun componente.

Impossibilità a lavorare in laboratorio perché un membro non ha condiviso il repository: -5 punti per ciascun membro.

La valutazione terrà conto anche della modularità e della chiarezza del codice (anche i commenti).
